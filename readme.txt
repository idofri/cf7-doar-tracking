Enables package delivery tracking using Israel Post & Contact Form 7.

== Description ==

The plugin adds A new CF7 form-tag called "doar tracking" which will force A form to function as package-tracking form based on the Israeli postal web-api.

= Features =
* Auto Bilingual - english / hebrew (depends on sites locale).

== Installation ==

= Installing CF7 Doar Tracking Plugin =

* Download the plugin zip file.
* Login to your WordPress Admin. Click on Plugins | Add New from the left hand menu.
* Click on the "Upload" option, then click "Choose File" to select the zip file from your computer. Once selected, press "OK" and press the "Install Now" button.
* Activate the plugin.

When activated, it will add A new CF7 form-tag called "doar tracking".
When added, the form will always be invalidated - the validation error will be replaced with relevant tracking information.
